## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

1️⃣ Git

Baixa i instal·la Git: https://git-scm.com/download/win

Durant la instal·lació, activa l’opció de “Git from the command line and also from 3rd-party software”.

Configura nom i correu:

git config --global user.name "Guillem Vázquez"
git config --global user.email "email@example.com"
git config --global core.autocrlf true

2️⃣ Node.js

Baixa la versió LTS de Node.js: https://nodejs.org/en/

Això instal·larà també npm, necessari per instal·lar dependències.

3️⃣ Editor de codi (opcional però recomanat)

VSCode: https://code.visualstudio.com/

També pots afegir extensions com Tailwind CSS IntelliSense per treballar còmodament.

4️⃣ Clonar el projecte

Un cop Git i Node instal·lats:

git clone https://github.com/USERNAME/REPO.git
cd REPO
npm install

5️⃣ Fitxer .env.local

Crea un fitxer .env.local amb les teves claus de Supabase (NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY).

Sense això, el projecte no funcionarà.

6️⃣ Executar la web
npm run dev


Accedeix a http://localhost:3000.
